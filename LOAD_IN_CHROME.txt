📦 HOW TO LOAD THIS EXTENSION IN CHROME (2 MINUTES)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 1: OPEN CHROME EXTENSIONS PAGE
────────────────────────────────────

   Method A: Type in address bar
   → chrome://extensions/
   → Press Enter

   Method B: Via menu
   → Click ⋮ (three dots, top-right)
   → More Tools → Extensions

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 2: ENABLE DEVELOPER MODE
──────────────────────────────

   → Look for "Developer mode" toggle (top-right corner)
   → Click to turn it ON (should turn blue)
   → New buttons will appear

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 3: LOAD THE EXTENSION
───────────────────────────

   → Click "Load unpacked" button
   → Navigate to this folder:
     "AUDIO ONLY FOR YOUTUBE™ - ELITE EDITION"
   → Click "Select Folder" or "Open"

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 4: VERIFY INSTALLATION
────────────────────────────

   ✅ Extension card appears
   ✅ Name: "Audio Only for YouTube™ - Elite Edition"
   ✅ Version: 1.0.0
   ✅ Toggle is ON (enabled)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 5: PIN TO TOOLBAR (OPTIONAL)
──────────────────────────────────

   → Click puzzle icon 🧩 in Chrome toolbar
   → Find "Audio Only for YouTube™ - Elite Edition"
   → Click pin icon 📌
   → Extension icon now visible in toolbar

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🎵 TEST IT NOW!

   1. Go to YouTube: https://youtube.com/watch?v=dQw4w9WgXcQ
   2. Click extension icon in toolbar
   3. Toggle "Audio-Only Mode" to ON
   4. Enjoy the elite audio experience! 🎧

🐛 TROUBLESHOOTING

   Problem: Extension not working?
   Solution: Refresh the YouTube page (F5)

   Problem: No audio playing?
   Solution: Check volume settings and video itself

   Problem: UI looks broken?
   Solution: Clear cache and refresh page
